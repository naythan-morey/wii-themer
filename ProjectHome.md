A tool to install Custom Themes via IOS236/249
Features:
- Name theme anything
- USB 2.0 support
- SD support
Requirements:
- Folder on SD/USB named "themes"
- Wii
- Wii/GC Controller
- USB or SD storage device
- HBC 1.0.8 or greater installed
- Based off of CleaRip by Emu\_Kidid
